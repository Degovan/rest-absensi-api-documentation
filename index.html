Rest Login

  • Untuk Rest login sendiri disini menggunakan route

    POST http://restabsen.wahyuartadigital.com/api/v2/login

    POST http://restabsen.wahyuartadigital.com/api/v2/login

        "code": 401,
        "success": false,
        "message": "error, doesn't pass validation",
        "data": {
            "old_value": {
                "email": null
            },
            "errors_validation": {
                "email": [
                    "email tidak boleh kosong"
                ],
                "password": [
                    "password tidak boleh kosong"
                ]
            }
        }
    }
  • Error, email & password don't match

    Apabila mengakses ke route login berikut

    POST http://restabsen.wahyuartadigital.com/api/v2/login

Rest Logout

  • Untuk Rest logout sendiri disini menggunakan route

    POST http://restabsen.wahyuartadigital.com/api/v2/logout

    Dan data yang wajib dikirimkan adalah api_token seperti berikut

        {
        "api_token" : "PZPmS8A8h5QbKCar1jyW4aJ5S4bJ3MJXSkQ5Wm3NwIo7OY7pnZNPzOI0hORc"
    }
Berikut Daftar Respon API dari akses ke REST LOGOUT
  • Success, logout action

    Apabila mengakses ke route logout berikut

    POST http://restabsen.wahyuartadigital.com/api/v2/logout

    dan mengirimkan data api_token yang cocok, maka akan mengembalikan respon seperti berikut

        {
        "code": 200,
        "success": true,
        "message": "successfully, the employee has been logged out and api_token has been removed"
    }

Rest Autentifikasi AUTH USER ( api_token )

  • Untuk Rest autentifikasi sendiri disini untuk setiap request wajib mengirimkan data api_token dari client

        {
        "api_token" : "PZPmS8A8h5QbKCar1jyW4aJ5S4bJ3MJXSkQ5Wm3NwIo7OY7pnZNPzOI0hORc"
    }

    Sebagai contoh disini kita mencoba mengambil data karyawan, yang dimana jika kita ingin mengakses route tersebut diwajibkan kita untuk login terlebih dahulu ( autentifikasi )

    GET http://restabsen.wahyuartadigital.com/api/v2/karyawan
Berikut Daftar Respon API dari akses ke REST AUTENTIFIKASI
  • Error, api_token is required

    Apabila mengakses ke route data karyawan berikut

    GET http://restabsen.wahyuartadigital.com/api/v2/karyawan

    tetapi tidak mengirimkan api_token, maka akan mengembalikan respon seperti berikut

        {
        "code": 401,
        "success": false,
        "message": "api token is required"
    }
